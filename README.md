# NBshare app
for Interaction Design ST 2016 LMU Munich

## Installation

- Install node.js
- Install grunt, bower and ionic
```
sudo npm i -g grunt-cli bower ionic
```

- Install dependencies
```
npm i
bower i
```

- Preview app in browser
```
grunt serve
```
